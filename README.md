## 🆙 개발 후기

### Unity/PM 락큼

 by PM

프로젝트의 부실한 기획만큼 개발자에게 무책임한 일이 없음을 깨닫게 되는 나의 첫 프로젝트였다. 

모든 기획이 완벽할 수는 없지만, 만약 다시 한번 프로젝트 기획 기회가 생긴다면 이번 경험을 밑거름 삼아 보다 나은 기획을 진행하고 싶다.

개발에 몰두해 PM으로서의 역할을 다하지 못했지만, 그래도 6주간 같이 개발을 진행해준 우리 개발자들 그리고 나와 제일 오래 런칭을 준비한 우리 디자이너에게 모두 감사하다. 나는 빡빡이다.

 by Unity developer

게임 앱을 개발해야 했기에 처음 접해보는 Unity로 앱의 Front와 게임기능을 구현했다.

Front: 

앱에 필요한 Scene 제작과 UI 구현 그리고 API 연동을 맡았다. 기기 해상도에 따른 autoScale이 되지 않아 UI 배치에 가장 골머리를 앓았다. (슬희가 다 해줌)

게임 개발: 

정적으로 만들어진 미로를 사용자에게 제공하는 것이 아닌, 동적으로 사용자가 제작한 미로를 공유, 플레이 할 수 있게 개발해야 했기에 많은 어려움이 있었다.

할말이 너무 많지만 다 할 수 없으니 pass.

### Unity/**Frontend Engineer** 서아

 by Frontend Engineer

이번 프로젝트는 실전 게임 제작을 향한 첫 디딤돌이었다. 또한 스토리에 따라 진행하는 것이 아닌 사용자와 인터렉션한 데이터를 이용한 것이라 더 많은 부분을 담당하지 못해 아쉬웠다.

프로토타입으로 된 디자인을 개발로 옮기는 것이 주요업무였다. 이 외에도 플레이어 컨트롤러 이동, 안드로이드 뒤로가기 등 게임의 보조적인 업무를 담당했다. 

다른 해상도와 비율도 모두 고려하며 배치하는 것도 그렇지만 바로 확인하기 어려운 프리팹 배치가 가장 난관이었다. 

좋은 경험, 결과물, 팀 구성원 들을 얻어가는 시간이었다. (팀장님, 팀원들 모두 수고했습니당)

### SpringBoot/Back**end Engineer** RillA_릴라

 by Backend Engineer

첫 협업을 통한 프로젝트는 설렘 반 두려움 반이였다. 분명 팀들에게 폐가되는 부분이 있을 거라 생각했기에, 두려운 부분은 존재했다. 하지만 좋은 팀원들과 진행된 Miner는 그 두려움을 없애주었다!

여러 server API를 만들기 위해 설계하는 과정에서 큰 어려움은 없었지만, 이렇게 설계하는 것이 효율적일 것이며, 이것보다 더 좋은 방법이 없을 까? 하는 의구심이 계속들었다. 따라서 클린 코드에 대한 관심이 하나도 없었는데 이번 프로젝트를 진행하면서 점차 관심을 가지게 된 거 같다.

생각보다 많은 경험과 완벽하진 않지만 우리의 뿌듯한 결과물을 Miner팀과 함께해서 좋았습니다!

팀장, 팀원님 수고많으셨습니다:)

### SpringBoot/Back**end Engineer** 서리 은수

### SpringBoot/Backend Engineer 루시 가현

by Backend Engineer

다른 사람들이 실제로 사용할 수 있는 앱을 개발하는 과정에 참여하는 것이 처음이라 기대가 되었다. 프로젝트를 진행하는 동안 힘들기도 했지만 내가 구현한 API들이 앱에서 필요한 기능대로 작동하는 것이 보이니까 재미있었던 부분이 컸다. 그리고 모르는게 있을 때에는 팀원들이 즉각적으로 알려주며 도움을 줘서 정말 고마웠다.

앱 개발만이 아니라 협업을 하는 경험까지 많은 것을 배울 수 있던 뜻깊은 기간이었다. 끝나지 않을 것 같던 프로젝트 기간이 마무리되고 앱도 완성이 되어 뿌듯하다. 누구 하나 대충하는 것 없이 모두가 노력한 결과라고 생각한다. 앱이 얼른 출시되어 사람들이 우리가 만든 게임을 할 생각에 설렌다.
